# QA Prompt Engineering Feladat

Ez a repository egy középiskolásoknak szóló QA (Quality Assurance) prompt engineering feladatot tartalmaz, valamint egy segédletet a CREATE keretrendszer használatához.

## Tartalom

- [Feladat.md](Feladat.md) – A megoldandó feladat leírása, követelmények és értékelési szempontok.
- [Segitseg.md](Segitseg.md) – Segédlet a CREATE keretrendszerhez és a hatékony promptíráshoz.

## Használat

Olvasd el a [Feladat.md](Feladat.md) dokumentumot, majd a [Segitseg.md](Segitseg.md) alapján készítsd el a megoldásodat!
